python_benchmark
===========

##What is it?

pythonで作られた簡易ベンチマーク。
大文字小文字のアルファベット＋0～9までの数字の合計62文字の中からランダムで10桁の文字を求めるプログラム。


推奨環境はlinux環境でpythonが必要です。
動作チェックはYosemiteのpython2.2です。


##How to Use

```
$ git clone https://github.com/flum1025/python_benchmark.git
$ cd python_benchmark
$ python python_benchmark.rb

```
